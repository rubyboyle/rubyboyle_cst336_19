React

Sketch adds a bunch of useless markup that you can remove. You should end up with something like the code to the left. Note addition classNames to the bounding box, and the heart path itself.

React is an efficient JavaScript library for building UIs. It lets you compose complex UIs from small and isolated pieces of code called components.





Notice how we can use those classNames we added to target individual shapes within the SVG. This lets us transition from white to red when the user hovers their mouse over the button.
